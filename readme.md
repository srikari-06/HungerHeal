# HungerHeal

A web application to combat hunger by connecting donors, NGOs, and organizations.

## Features

- Donor registration and donation tracking
- NGO and organization management
- Reward system for donors
- Dashboard for monitoring activities

## Getting Started

1. Clone the repository
2. Open index.html in your browser
